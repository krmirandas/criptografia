El codigo con el que resolvimos algunos de los ejercicios se encuentran dentro de la carpeta src.

Para ejecutarlo no es necesario tener instalado alguna biblioteca externa.

Para ejecutarlos, simplemente debe anteponerse el comando python3 y a continuacion el nombre del archivo a ejecutar. 

En caso del archivo imageDecipher.py, la salida estara en la carpeta vacia images/ dentro de src
